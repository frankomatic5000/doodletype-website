# DoodleType 🖍️

> Original characters. Real personalities. Designed by a 12-year-old who refused to draw boring stuff.

**Live site:** https://doodletype.com  
**Shop:** https://doodletype.myshopify.com

## Stack
- Vanilla HTML/CSS/JS — no frameworks, fast & simple
- Deployed via Vercel (auto-deploys on push to `main`)
- Shop powered by Shopify + Printify (print on demand)
